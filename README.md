## Modelagem Computacional da Dinâmica de Transmissão do Sarampo: Uma Abordagem via Modelo SIR

Repositório destinado para a disciplina Modelagem Matemático-Computacional Aplicada à epidemiologia - 2025.2

## Requisitos
Para o pleno funcionamento deste código é necessário instalar:
- Python 3
- numpy
- matplotlib

## Execução

### 1. Clone o repositório
```bash
git clone [https://github.com/Redoze/Epidemiology-Computing-2025.2.git](https://github.com/Redoze/Epidemiology-Computing-2025.2.git)
cd Epidemiology-Computing-2025.2
```

### 2. Crie um ambiente virtual
```bash
# No Windows
python -m venv venv
.\venv\Scripts\activate

# No Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 3. Instale as dependências
```bash
pip install -r requirements.txt
```
### 4. Execute
```bash
python SIR.py
```
